# Laravel React Dashboard

A full-stack application example with a Laravel PHP APIs and a React frontend

## Requirements

- PHP 7.4
- MySQL
- Composer
- Docker

## Installation

    composer install
    
## Docker

    docker compose build
    docker compose up

Access the admin API container:

    docker exec -it laravel-admin_backend_1 sh

## Laravel

    php artisan serve
    
## Run tests

    php artisan test
