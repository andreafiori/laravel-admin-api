# Laravel React Dashboard

A full-stack application with a Laravel PHP APIs and a React frontend

## Requirements

- PHP 7.4
- MySQL
- Composer

## Installation

    composer install
    
## Docker

    docker compose build
    docker compose up

## Laravel

    php artisan serve
    
## Run tests

    php artisan test

## Technologies

- Laravel 8.x
    - Laravel IDE helper
    - Laravel passport
- Open API
- Docker
- React

## Tools

- PHPStorm IDE
